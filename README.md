# DataScienceIntern

This repository contains the completed data science tasks for the Oasis Infobyte Internship Program.

### Completed Tasks
- **Task 1**: Iris Flower Classification (`K_Venkatesh_Hebbar_Task1.ipynb` / `app_task1.py`)
- **Task 2**: Unemployment Analysis with Python (`K_Venkatesh_Hebbar_Task2.ipynb` / `app_task2.py`)
- **Task 3**: Car Price Prediction with Machine Learning (`K_Venkatesh_Hebbar_Task3.ipynb` / `app_task3.py`)
- **Task 4**: Email Spam Detection (`K_Venkatesh_Hebbar_Task4.ipynb` / `app_task4.py`)
- **Task 5**: Sales Prediction Using Python (`K_Venkatesh_Hebbar_Task5.ipynb` / `app_task5.py`)

All notebooks contain exploratory data analysis, visualisations, and trained machine learning models according to the requirements. The Python scripts (`app_taskX.py`) can be used to deploy interactive web applications via Streamlit.
